# Changelog

## [1.0.0] — 2026-06-22

### 初始发布

首次发布 Basalt OS 清算分账系统知识库 Skill，覆盖以下 11 个知识模块：

| 模块 | 文件 | 内容范围 |
|---|---|---|
| 账户体系 | `accounts.md` | 5 类账户定义、前缀规则、开户方式、余额查询、资金流向 |
| 商户入驻 | `onboarding.md` | 平台/商户入驻流程、预入驻+激活、POS 绑定、状态流转 |
| 清算流程 | `clearing.md` | 12 步全流程、订单上送、清算文件、批次提交、资金归集 |
| 付款与协议 | `payments.md` | 协议代扣/授权代付对比、签约/解约、付款单状态 |
| 退款与退保 | `refunds.md` | 扣保退保 4 步流程、deposit.notify 5 种类型、退款规则 |
| 提现 | `withdrawal.md` | 提现两步流程、费率回收机制、银行卡绑定 |
| 对账 | `reconciliation.md` | 对账流程、提现费率对账 5 步、回单下载 |
| 外部同步 | `external-sync.md` | 10 个同步点、通知+查询双保险、幂等处理 |
| 费率 | `fees.md` | 各业务费率、承担方、计费示例 |
| 角色权限 | `roles-permissions.md` | 角色定义、操作范围、数据可见性、商户限权 |
| 问题排查 | `troubleshooting.md` | 常见错误、排查步骤、应急处理、错误码 |

### 已知限制

- 仅做知识查询，不包含数据读写操作
- 角色权限模块部分内容基于业务流程推断，已标注【推断】
- 三联 ERP 同步细节在 PRD 中缺失，以系统方案为准
